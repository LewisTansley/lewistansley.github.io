SMPL Towers

A Run Down on My active project 'SMPL Towers'

SMPL Towers is an experimental tower defence game, designed with simplicity, minimalism and heavy use of strategy.

The game is developed within the unity engine, making heavy use of it's universal User Interface system, this allows the game to work across multiple platforms with no additional overhead required by the engine and during development.

It is designed with mobile and pc platforms in mind, with a browser version coming later down the line in development.
